## 装机必备常用环境配置，不断更新中

### 1.[pip](https://pip.pypa.io/en/stable/)
#### Install
On Debian and Ubuntu:
```
sudo apt-get install python-pip
```
#### Upgrade
On Linux or OS X:
```
pip install -U pip
```
